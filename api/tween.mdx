`tween()` is a thin convenience wrapper around Roblox's `TweenService:Create()`. It creates the tween, calls `:Play()` immediately, and returns the `Tween` object so you can optionally cancel it, pause it, or inspect its playback state. You pass the same arguments you would give `TweenService:Create()` ‚Äî no extra setup required.

## Signature

```lua
AnimationService:tween(instance, info, goal) --> Tween
```

## Parameters

<ParamField path="instance" type="Instance" required>
  The Roblox `Instance` whose properties you want to animate. This can be any
  instance type that exposes tweenable properties ‚Äî `BasePart`, `Model`,
  `Frame`, `ImageLabel`, and so on.
</ParamField>

<ParamField path="info" type="TweenInfo" required>
  A `TweenInfo` value that controls how the animation plays. This includes:
  - **duration** ‚Äî total time in seconds
  - **EasingStyle** ‚Äî e.g. `Enum.EasingStyle.Quad`
  - **EasingDirection** ‚Äî `In`, `Out`, or `InOut`
  - **RepeatCount** ‚Äî number of times to repeat (`-1` for infinite)
  - **Reverses** ‚Äî whether to play back in reverse after each cycle
  - **DelayTime** ‚Äî seconds to wait before the tween starts
</ParamField>

<ParamField path="goal" type="table" required>
  A table mapping property names (as strings) to their target values. For
  example, `{ Transparency = 1 }` or `{ Position = Vector3.new(10, 5, 0) }`.
  Every key must be a property that `TweenService` supports tweening.
</ParamField>

## Returns

Returns a `Tween` object. The tween is already playing when it is returned.
Call `:Cancel()` on it to stop early, `:Pause()` to pause, or read
`.PlaybackState` to check whether it has completed.

## Example

```lua
-- Fade a part to fully transparent over 1 second
local info = TweenInfo.new(1, Enum.EasingStyle.Quad, Enum.EasingDirection.Out)
Anim:tween(part, info, { Transparency = 1 })

-- Move a part to a target position in 0.5 seconds
Anim:tween(part, TweenInfo.new(0.5), { Position = Vector3.new(10, 5, 0) })

-- Change color smoothly over 0.3 seconds
Anim:tween(part, TweenInfo.new(0.3), { Color = Color3.fromRGB(255, 0, 0) })

-- Store the tween to cancel it later
local t = Anim:tween(part, TweenInfo.new(5), { Transparency = 1 })
task.delay(2, function()
    t:Cancel()
end)
```

<Tip>
  Store the returned `Tween` if you need to interrupt the animation. Call
  `:Cancel()` to stop it immediately, or check `tween.PlaybackState` to see
  whether it is still running, paused, or has completed.
</Tip>

<Note>
  Any property that Roblox's built-in `TweenService` can animate is valid in
  the `goal` table ‚Äî this includes numeric types, `Vector3`, `CFrame`,
  `Color3`, `UDim2`, and more. Unsupported property types will cause
  `TweenService` to throw an error at creation time.
</Note>
